# Age Gate Block

## Overview
Full-screen age verification overlay for 18+ storefront access.

## Authoring
Add an `age-gate` block and configure with label/value rows:

- `Headline` | Heading text
- `Message` | Main message text
- `Legal Copy` | Optional legal line
- `Confirm Label` | Confirm button label
- `Deny Label` | Exit button label
- `Exit URL` | Redirect URL when denied
- `Min Age` | Display-only age value (default `18`)
- `Storage Key` | Optional localStorage key
- `Background Image` | Optional image
