# Social Media Carousel Block

## Overview
Carousel block for mixing Instagram posts and YouTube Shorts cards.

## Authoring
Create one row per card with these columns:

- Column 1: Platform label (`Instagram` or `YouTube Shorts`)
- Column 2: Post URL (link)
- Column 3: Optional caption (rich text)

Supported URL formats:
- Instagram post/reel URLs from `instagram.com`
- YouTube Shorts URLs (`youtube.com/shorts/...`)
- YouTube watch/embed URLs and `youtu.be` URLs
